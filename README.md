# Addu-sh

A custom Unix shell built from scratch in C.

## About
Addu-sh is a fully hand-written Unix shell implementing lexing, parsing,
and execution of commands without relying on any shell libraries.

## Build
```
make
```

## Run
```
./addu-sh
```
